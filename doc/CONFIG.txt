usage: msiem config [-h] [--print] [--set '<section>' ['<option>' '<value>']
                    ['<section>' ['<option>' '<value>'] ...]]

    Set and print your msiempy config.  
    

optional arguments:
  -h, --help            show this help message and exit
  --print               Print configuration fields, password base 64 included. (default: False)
  --set '<section>' ['<option>' '<value>'] ['<section>' ['<option>' '<value>'] ...]
                        Set the config option to the specified value if passed (can be repeated), OR
                        inveractively prompt for specified configuration section: 'esm' or 'general'.
                        (default: [])
